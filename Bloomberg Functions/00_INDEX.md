## How to Use This Guide

- **Command format**: All Bloomberg commands are written in `UPPERCASE MONOSPACE`.
- **Security-level functions** require a security to be loaded first (e.g., `AAPL US Equity` then `FA`).
- **Market-level functions** work without a loaded security (e.g., `WEI`).
- **Sub-screens** are tabs or pages within a function, accessed by number or menu.

## Quick Glossary

| Term | Meaning |
|------|---------|
| `<EQUITY> GO` | Press the yellow Equity key, then GO (Enter) |
| Amber key | Yellow sector keys on the keyboard (Equity, Govt, Corp, etc.) |
| Launchpad | Customizable dashboard workspace |
| FLDS | Field search — look up any data field by keyword |
