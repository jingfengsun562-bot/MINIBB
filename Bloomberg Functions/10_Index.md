# 10 — Index Analysis

Functions for equity and fixed income benchmark indices, custom index construction, and constituent analysis.

---

## Index Overview & Monitoring

### `WEI`
**Description**: World Equity Indices — real-time global equity index monitor.  
**Key sub-screens**:
- Regional groupings (Americas, EMEA, Asia-Pacific)
- Performance (1D, 1W, 1M, QTD, YTD, 1Y)
- Valuation (P/E, P/B, Div Yield, Earnings Yield)
- Market cap and breadth
- Futures basis / premium

### `WGBI`
**Description**: World Government Bond Index — global sovereign bond index performance and analytics.

### `BI FI` (Bloomberg Indices — Fixed Income)
**Description**: Bloomberg Fixed Income Index family dashboard (Bloomberg Aggregate, US Corporate, Global HY, EM, etc.).

---

## Index Constituents & Analytics

### `MEMB`
**Description**: Index Members — full constituent list for any index with weights, contribution, and fundamental data.  
**Key sub-screens**:
- Constituent list with weights
- Sector/country breakdown
- Top/bottom contributors to return
- Sort by weight, return, market cap
- Export full list

### `INDX`
**Description**: Index Description — key facts about an index (methodology, rebalance frequency, number of constituents, base date).

### `IMAP`
**Description**: Industry Map — interactive industry classification tree (GICS or BICS) showing sector → industry group → industry → sub-industry with performance.

### `GIP`
**Description**: Generic Index Price — continuous historical price chart for any index.

---

## Index Performance & Attribution

### `TRA`
**Description**: Total Return Analysis — historical total return, price return, and dividend return for an index with risk metrics.  
**Key sub-screens**:
- Cumulative return chart
- Annualized return & volatility
- Sharpe ratio, Sortino ratio
- Drawdown analysis
- Calendar year returns

### `COMP`
**Description**: Comparison — overlay multiple indices to compare performance, risk, and characteristics.

### `ATTR`
**Description**: Attribution — return attribution by sector, country, or factor for an index over a chosen period.

---

## Custom Indices

### `CIXB`
**Description**: Custom Index Builder — create custom equity or fixed income indices with user-defined rules (universe, weighting, rebalance, filters).  
**Key sub-screens**:
- Universe selection
- Weighting scheme (market cap, equal, fundamental, custom)
- Rebalance frequency
- Filter criteria (min market cap, liquidity, sector caps)
- Backtest with historical performance
- Export to PORT or Excel

### `BSKT`
**Description**: Basket Tool — create and track custom baskets of securities with real-time P&L.

---

## Factor & Style Indices

### `FSRC`
**Description**: Factor Search — explore factor index families (Value, Growth, Momentum, Quality, Low Vol, Size) with performance and composition.

### `PRMS`
**Description**: Factor Premia Monitor — track factor returns across regions and time periods.
