# 01 — General & Navigation

Core terminal commands for search, communication, settings, and general workflow.

---

## Terminal Navigation

### `HELP` / `F1`
**Description**: Context-sensitive help. Press once for help on current screen; press twice to reach Bloomberg Help Desk (live chat with support).

### `BLP`
**Description**: Bloomberg Launchpad — customizable multi-monitor workspace with drag-and-drop components (charts, watchlists, news).  
**Key sub-screens**:
- Component library (charts, monitors, news panels)
- Layout templates (pre-built workspaces by role)
- Shared workspaces (team collaboration)

### `GRAB`
**Description**: Screenshot tool. Captures current screen and allows annotation, saving, or sharing via MSG.

### `LOG`
**Description**: Activity log of all functions run in the current session. Useful for retracing steps.

### `PDF`
**Description**: Print/export current screen to PDF.

---

## Search & Discovery

### `BI`
**Description**: Bloomberg Intelligence — proprietary research covering industries, economics, litigation, and government. Analyst dashboards with data-driven insights.  
**Key sub-screens**:
- Industry analysis (sector dashboards)
- Economic research (country/region)
- Litigation analytics
- Government analysis

### `SECF`
**Description**: Security Finder — universal search across all asset classes. The starting point when you don't have a ticker.  
**Key sub-screens**:
- Asset class filter (Equity, Fixed Income, Fund, Index, Commodity, Currency, Govt)
- Geography, exchange, sector filters
- Custom criteria builder
- Results export

### `FLDS`
**Description**: Data Field Search — look up Bloomberg field mnemonics by keyword. Essential for Excel API and data scripting.  
**Key sub-screens**:
- Keyword search
- Category filter (Fundamentals, Pricing, Analytics, etc.)
- Field definition & data type
- Override parameters

### `FIND`
**Description**: Quick-search bar. Type any security name, ticker, ISIN, CUSIP, SEDOL, or keyword to locate it.

### `DOCA`
**Description**: Bloomberg document search. Searches across filings, transcripts, presentations, and research.

---

## Communication & Messaging

### `MSG`
**Description**: Bloomberg Messaging — instant messaging system between Bloomberg Terminal users. Industry-standard for buy-side/sell-side communication.  
**Key sub-screens**:
- Compose / inbox / sent
- Contact directory (PEOPLE)
- Chat rooms (IB chats)
- File sharing & screenshots

### `IB`
**Description**: Instant Bloomberg — real-time chat with Bloomberg users. Supports group chats for trade ideas and deal flow.  
**Key sub-screens**:
- Active chats
- Chat history
- Blast messages (one-to-many)

### `PEOP` / `PEOPLE`
**Description**: People directory — search for Bloomberg Terminal users by name, firm, or role. Used for networking and finding counterparties.

---

## Alerts & Monitoring

### `ALRT`
**Description**: Alert Manager — set price, news, economic, or custom data alerts. Notifications via terminal, email, or mobile.  
**Key sub-screens**:
- Price alerts (level, % change, volume)
- News alerts (keyword, security-linked)
- Economic release alerts
- Custom condition builder
- Delivery settings (terminal pop-up, email, Bloomberg Anywhere)

### `BALT`
**Description**: Bloomberg Alert Templates — pre-built alert configurations by workflow.

---

## User Settings & Customization

### `SPDL`
**Description**: Speed Dial — bookmark favorite functions for quick access (appears on terminal home screen).

### `BSET`
**Description**: Bloomberg Settings — configure defaults for number format, time zone, display, colors, and keyboard.

### `TSET`
**Description**: Trading System Settings — configure order routing, default brokers, and trading preferences.

### `LPHD`
**Description**: Launchpad Home Directory — manage and organize Launchpad workspaces.

---

## Data Export & Connectivity

### `DAPI`
**Description**: Data API overview — documentation for Bloomberg's programmatic data access (B-PIPE, Server API, Desktop API).

### `WAPI`
**Description**: Excel/VBA API documentation and function builder. Interactive tool to construct BDP/BDH/BDS formulas.  
**Key sub-screens**:
- Function builder (BDP, BDH, BDS)
- Field browser
- Override parameters
- Sample formulas with copy-paste

### `BSRCH`
**Description**: Bloomberg Search — structured search across all Bloomberg data with exportable results. Useful for building universes.
